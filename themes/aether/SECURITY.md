# Security Policy

Aether is maintained as part of the main site repository, [DivinerHJF/hugo-blog-Aether](https://github.com/DivinerHJF/hugo-blog-Aether), and does not publish separate supported theme versions.

If you need to report a security concern related to this site/theme repository, please use the main repository issue tracker or contact the site maintainer through [philohao.com](https://philohao.com/).

Aether is adapted from [DoIt](https://github.com/HEIGE-PCloud/DoIt); upstream copyright and license notices remain preserved.
