# Aether 本站改造记录

Aether 是 [philohao.com](https://philohao.com/) 的自用 Hugo 主题，随主仓库一起维护，不再按独立主题版本发布。

## 维护原则

- 以本站内容展示、样式调整、构建稳定性为优先目标。
- 不再维护 `exampleSite`、主题商店截图、独立主题贡献流程或单独的安全版本矩阵。
- 主题变更记录以主仓库 Git 提交历史为准；重要说明会同步写入主仓库 `README.md`。

## 来源说明

本主题基于 [DoIt](https://github.com/HEIGE-PCloud/DoIt) 改造，并保留 DoIt 及其上游项目的版权与许可证声明。

旧版上游 DoIt 的版本化 Changelog 不再复制维护，避免误以为 Aether 仍跟随 DoIt 的发布节奏。如需查询上游历史，请参考 DoIt 仓库的发布记录与提交历史。
