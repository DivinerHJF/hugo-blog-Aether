# Aether Theme

Aether is a self-use Hugo theme for [philohao.com](https://philohao.com/), adapted from [DoIt](https://github.com/HEIGE-PCloud/DoIt).

The theme is maintained together with the main repository, [DivinerHJF/hugo-blog-Aether](https://github.com/DivinerHJF/hugo-blog-Aether), and is no longer intended to be installed as a standalone theme or Git submodule.

For details, see the Chinese README: [README.zh-cn.md](README.zh-cn.md).
